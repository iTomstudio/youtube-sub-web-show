## 🔧 技术实现

### 使用的技术
- Vue 3 Composition API
- Tailwind CSS（响应式设计）
- lucide-vue-next 图标库
- 动态渐变色系统
- 平滑滚动交互

### 响应式设计
- **移动端**（< 640px）：单列布局
- **平板**（640-1024px）：2列服务卡片、2列联系方式
- **桌面**（> 1024px）：2列服务卡片、3列联系方式、4列示例

### 深色模式支持
- 所有渐变都有深色模式变体
- 文字颜色自动适配
- 卡片背景和边框适配
