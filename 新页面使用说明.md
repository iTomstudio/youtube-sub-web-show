## 页面功能介绍

### 1. 免费示例页面 (`/examples`)

**功能特点**：
- 网格布局展示 6 个视频示例
- 每个视频卡片包含：
  - 封面图片（16:9 比例）
  - 视频标题
  - 简短描述
  - 难度标签（N5-N1）和主题标签
  - 两个下载按钮：百度网盘（蓝色）和夸克网盘（紫色）
- 响应式设计：
  - 移动端：1 列
  - 平板端：2 列
  - 桌面端：3 列
- 深色模式完全支持

**当前示例数据**：
- Day 240: 婚礼讲话 vlog
- Day 239: 日本圣诞节
- Day 238: 浅草散步
- Day 237: 草津温泉
- Day 236: 博物馆购物
- Day 235: 冬季购物分享

### 2. 定制服务页面 (`/services`)

**功能特点**：
- 服务介绍部分（上半部分）：
  - 日语资源代查服务（带搜索图标）
  - 电子书查找服务（带书籍图标）
  - 每个服务列出 5 个特点
- 联系方式部分（下半部分）：
  - 微信二维码卡片

  - 居中对齐，易于扫描
- 服务流程说明（4 步流程）
- 完全响应式，支持深色模式

## 需要您完成的步骤

### 步骤 1：添加二维码图片

需要将您的微信和QQ二维码图片放到以下位置：

```
public/
└── qr-codes/
    ├── wechat-qr.png  ← 请添加您的微信二维码
    └── qq-qr.png      ← 请添加您的QQ二维码
```

**图片要求**：
- 尺寸：建议 500x500 像素或更高
- 格式：PNG（可以有透明背景）
- 确保二维码清晰，可以正常扫描

**如何获取二维码**：
- 微信：微信 -> 我 -> 个人信息 -> 我的二维码 -> 保存图片
- QQ：QQ -> 头像 -> 个人资料 -> 我的二维码 -> 保存图片

### 步骤 2：添加视频封面图片（可选）

如果您想显示真实的视频封面，请将图片添加到：

```
public/
└── examples/
    ├── day240.jpg  ← Day 240 的封面图
    ├── day239.jpg  ← Day 239 的封面图
    ├── day238.jpg
    ├── day237.jpg
    ├── day236.jpg
    └── day235.jpg
```

**图片要求**：
- 尺寸：1280x720 或 640x360（16:9 比例）
- 格式：JPG 或 WebP
- 大小：建议每张不超过 500KB

**注意**：如果不添加图片，页面会显示渐变色背景的占位符，不影响页面功能。

### 步骤 3：更新下载链接

打开文件 [src/pages/FreeExamplesView.vue](src/pages/FreeExamplesView.vue)，找到 `examples` 数组（第 15-75 行），将下载链接从 `'#'` 替换为真实的网盘链接：

```typescript
// 示例：
{
  id: 1,
  title: '【Day 240】作为朋友代表在婚礼上的讲话vlog',
  thumbnail: '/examples/day240.jpg',
  description: '参加朋友婚礼，作为代表致辞的完整记录',
  tags: ['N5', 'N4', 'N3', 'N2', '在日生活', '赴日留学', '生活方式', '当地文化'],
  baiduLink: 'https://pan.baidu.com/s/你的真实链接',  // ← 修改这里
  quarkLink: 'https://pan.quark.cn/s/你的真实链接'   // ← 修改这里
}
```

### 步骤 4：（可选）更新联系方式文字

如果您想在二维码下方显示微信号或QQ号，可以编辑 [src/pages/CustomServicesView.vue](src/pages/CustomServicesView.vue)：

找到 `contactMethods` 数组（第 45-60 行），添加 `accountId` 字段：

```typescript
{
  type: 'wechat',
  qrCode: '/qr-codes/wechat-qr.png',
  label: '微信联系',
  hint: '扫码添加微信',
  accountId: 'your_wechat_id'  // ← 添加您的微信号
},
{
  type: 'qq',
  qrCode: '/qr-codes/qq-qr.png',
  label: 'QQ联系',
  hint: '扫码添加QQ',
  accountId: '123456789'  // ← 添加您的QQ号
}
```

## 如何添加更多视频示例

1. 打开 [src/pages/FreeExamplesView.vue](src/pages/FreeExamplesView.vue)
2. 在 `examples` 数组中添加新对象：

```typescript
{
  id: 7,  // 使用下一个递增的数字
  title: '【Day XXX】您的视频标题',
  thumbnail: '/examples/dayXXX.jpg',  // 对应的封面图片路径
  description: '视频的简短描述（1-2句话）',
  tags: ['N5', 'N4', '在日生活', '其他标签'],  // 相关标签
  baiduLink: 'https://pan.baidu.com/s/xxx',  // 百度网盘分享链接
  quarkLink: 'https://pan.quark.cn/s/xxx'   // 夸克网盘分享链接
}
```

3. 如果有封面图，将其保存到 `public/examples/` 目录
