# 二维码图片说明

## 需要的文件

请将以下二维码图片放在此目录下：

1. **wechat-qr.png** - 微信二维码
   - 建议尺寸：500x500 像素或更高
   - 格式：PNG（支持透明背景）
   - 确保二维码清晰可扫

2. **qq-qr.png** - QQ二维码
   - 建议尺寸：500x500 像素或更高
   - 格式：PNG（支持透明背景）
   - 确保二维码清晰可扫

## 注意事项

- 图片文件名必须完全匹配上述名称
- 建议使用高分辨率图片以确保在高清屏幕上显示清晰
- 如果使用透明背景，页面会自动适配深色模式
- 测试时请确保二维码可以正常扫描

## 如何生成二维码

可以使用以下工具生成二维码：
- 微信：微信 -> 我 -> 个人信息 -> 我的二维码 -> 保存图片
- QQ：QQ -> 头像 -> 个人资料 -> 我的二维码 -> 保存图片
